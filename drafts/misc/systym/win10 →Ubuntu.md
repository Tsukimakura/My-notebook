#### 3. 使用Rufus等工具制作启动U盘
[Rufus下载](https://rufus.ie/zh/)
- 注1：U盘将被格式化，提前备份重要数据，最好使用空U盘
- 注2：ISO映像文件不应存储于目标U盘内部(如果弹出`错误：不能指派盘符`可以检查这一条)
- 注3：选择ISO文件后其它参数选项不应该随意更改，除非你知道自己在做什么
- 注4：以管理员身份运行Rufus
- 关于持久分区：

| 概念                     | 说明                                           | 是否适合你的场景                         |
| ---------------------- | -------------------------------------------- | -------------------------------- |
| **持久分区 (Persistence)** | 在U盘运行的Ubuntu试用环境中，保存系统设置、安装的软件和文件。重启后更改不会丢失。 | **不适合**，你的目标是**将Ubuntu安装到电脑硬盘**。 |

- 启动U盘的原理：
1. **硬件层面：BIOS/UEFI 的引导**
    - 当你按下电脑电源键时，第一个被唤醒的不是硬盘上的Windows，而是主板上一块叫**BIOS**（或更新的**UEFI**）的微型系统。
    - BIOS/UEFI 的职责是进行硬件自检，然后按照预设的“启动菜单”顺序（比如：第一项硬盘，第二项光驱，第三项U盘）去寻找一个可以启动的操作系统。
    - 当你插入制作好的启动U盘，并在BIOS中将它设为第一启动项后，BIOS/UEFI 就会去读取U盘里的内容。
2. **软件层面：引导程序与临时系统**
    - 启动U盘里不仅仅有Linux系统文件（.iso解压后的内容），更重要的是它包含了一个**引导程序**（通常是 **GRUB**）。
    - BIOS/UEFI 把控制权交给这个引导程序，然后引导程序会找到U盘里的Linux内核和临时文件系统，并将它们加载到电脑的**内存（RAM）** 中运行。

#### 4.进入BIOS/UEFI
1. 完全关闭电脑
2. 插入制作好的Ubuntu启动U盘
3. 进入BIOS/UEFI
	开机时连续按特定键，参考
	 - Dell/联想：F12
	 - HP：F9或Esc
	 - ASUS：F2
	 - Acer：F2
4. 在`Boot`选项卡中选择U盘为启动项进入

#### 5. 安装Ubuntu
1. 从U盘启动后将进入GRUB菜单
2. 选择`Ubuntu`
3. 等待系统加载到试用桌面环境
	[[misc#如果在进入Ubuntu使用桌面时一直黑屏怎么办 | 如果加载很久一直黑屏怎么办]]
4. 双击桌面上的`Install Ubuntu`图标
5. 根据引导进行安装
6. 安装完成后点击restart now后**根据提示**拔掉U盘并按`Enter`
7. 基础系统更新、软件安装等后续操作
