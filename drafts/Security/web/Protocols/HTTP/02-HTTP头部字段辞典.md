- 以表格形式记录，包含：
    
    - 头部字段名
        
    - 所属类型（请求/响应/通用）
        
    - 功能描述
        
    - **CTF中的常见利用方式**（这是最重要的列！）
        
- 例如：

| 字段名          | 类型  | 功能     | CTF利用             |
| ------------ | --- | ------ | ----------------- |
| `Cookie`     | 请求  | 携带会话信息 | 会话固定、越权、伪造身份      |
| `Set-Cookie` | 响应  | 设置会话信息 | HttpOnly标记缺失导致XSS |
