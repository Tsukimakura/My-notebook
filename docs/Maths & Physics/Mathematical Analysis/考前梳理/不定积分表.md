### 常用不定积分表

#### 一、 基础幂函数与常数
| 序号  | 积分公式                                              | 备注               |
| :-- | :------------------------------------------------ | :--------------- |
| 1   | $\int 0 \, dx = C$                                | $C$ 为积分常数        |
| 2   | $\int 1 \, dx = x + C$                            |                  |
| 3   | $\int x^\mu \, dx = \frac{1}{\mu+1}x^{\mu+1} + C$ | 条件：$\mu \neq -1$ |
| 4   | $\int \frac{1}{x} \, dx = \ln \|x\| +C$           |                  |
| 5   | $\int \frac{1}{x-a} \, dx = \ln \|x-a\|+C$        |                  |

#### 二、 指数函数
| 序号 | 积分公式 | 备注 |
| :--- | :--- | :--- |
| 6 | $\int e^x \, dx = e^x + C$ | |
| 7 | $\int a^x \, dx = \frac{a^x}{\ln a} + C$ | 条件：$a>0, a \neq 1$ |

#### 三、 三角函数
| 序号 | 积分公式 | 备注 |
| :--- | :--- | :--- |
| 8 | $\int \cos x \, dx = \sin x + C$ | |
| 9 | $\int \sin x \, dx = -\cos x + C$ | 注意负号 |
| 10 | $\int \frac{1}{\cos^2 x} \, dx = \int \sec^2 x \, dx = \tan x + C$ | |
| 11 | $\int \frac{1}{\sin^2 x} \, dx = \int \csc^2 x \, dx = -\cot x + C$ | 注意负号 |

#### 四、 代数分式与反三角相关
| 序号  | 积分公式                                                                            | 备注     |
| :-- | :------------------------------------------------------------------------------ | :----- |
| 12  | $\int \frac{1}{1+x^2} \, dx = \arctan x + C$                                    |        |
| 13  | $\int \frac{1}{\sqrt{1-x^2}} \, dx = \arcsin x + C$                             |        |
| 14  | $\int \frac{1}{\sqrt{x^2 \pm a^2}} \, dx = \ln \| x + \sqrt{x^2 \pm a^2} \| +C$ | **重要** |
